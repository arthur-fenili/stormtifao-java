StormTifão - Global Solution

Solução integrada para gestão de desastres naturais

Este projeto foi desenvolvido para o Global Solution FIAP e simula uma plataforma de apoio à população e autoridades em situações de eventos extremos como tempestades e furacões.

Funcionalidades:
  • Cadastro e listagem de pessoas desaparecidas durante desastres.
  • Consulta de abrigos seguros integrados a outras aplicações (mobile e .NET).
  • Dashboard de previsão de risco de tempestade ou furacão utilizando inteligência artificial sobre dados reais de clima.
  • Interface amigável, responsiva e multilíngue (PT/EN).
  • Autenticação via OAuth2 (Google) para maior segurança.
Objetivo:
Facilitar o registro de desaparecidos, informar sobre locais seguros e fornecer alertas inteligentes, ajudando a salvar vidas e otimizar recursos em momentos críticos.
